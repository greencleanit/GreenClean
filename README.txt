Inserimento anagrafica Nuovo Cliente

creazione di un'interfaccia per l'inserimento dei dati di un nuovo cliente.
Lo strumento deve essere raggiungibile da chiunque attraverso un link, al termine della compilazione l'utente deve vedere una schermata di esito e ringraziamento per la collaborazione con il tasto chiudi.
Il sistema genera una mail per ogni inserimento di modulo
ci sono sei problemi funzionali e una parte di codice è visibile sopra il modulo
